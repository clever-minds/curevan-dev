# **App Name**: Curevan

## Core Features:

- Therapist Discovery: Allow users to view and filter healthcare providers on a map within a 5km radius.
- Appointment Booking: Enable patients to book appointments with selected therapists.
- Booking Management: Permit therapists to view and manage appointment bookings.
- PCR Form: Facilitate the creation and digital signing of Patient Care Reports (PCR).
- User Onboarding: Allow users to onboard, including agreeing to location sharing.
- AI-assisted notes: Enable therapists to automatically create objective notes based on patient description of problem, using a gen AI tool.

## Style Guidelines:

- Primary color: A gradient from dark blue-purple (#4A23D1) to teal (#2AB8A6), mirroring the logo's color scheme, to represent innovation and health.
- Background color: Light gray (#F7F7F7).  A neutral background provides a clean backdrop for content.
- Accent color: Teal (#2AB8A6) from the logo. This color highlights interactive elements and calls to action.
- Body and headline font: 'Nunito Sans' - a modern, rounded sans-serif font to give a friendly and approachable feel.
- Use modern, line-style icons with rounded corners to complement the font and overall aesthetic.
- Maintain a clean and intuitive layout for easy navigation and readability, with ample spacing and clear visual hierarchy.
- Use subtle animations for loading states and transitions to enhance user experience and provide feedback.
